Privacy Policy

This Privacy Policy explains how we handle user data when using our WhatsApp automation service.

Information We Collect

We do not store personal data unless provided explicitly by the user. Any data processed is used only to deliver the requested service.

How We Use Data

Data is used strictly for responding to user messages and executing workflows triggered by user inputs.

Data Storage

No data is permanently stored. Temporary data used during workflow execution is deleted automatically.

We use the WhatsApp Business Cloud API and n8n automation platform. These services may temporarily process data in order to deliver responses.
